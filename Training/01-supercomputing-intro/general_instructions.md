# General instructions for exercises

Log in to Mahti:

```
ssh my-username@mahti.csc.fi
```

Go into SCRATCH directory and create a personal workspace:

```
cd /scratch/project_2016753
mkdir -p $USER
cd $USER
```

Clone this repository:

```
git clone git@github.com:cschpc/isc-student-cluster-competition-2026.git
```

Information about CSC supercomputer environments:
- [Module system](https://docs.csc.fi/computing/modules/)
- [Batch job system](https://docs.csc.fi/computing/running/creating-job-scripts-puhti/)
